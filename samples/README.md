# Sample Financial Statements

This directory contains sample/redacted financial statements for testing the VWB system.

## Files

- `income_statement_sample.xlsx` - Sample income statement
- `balance_sheet_sample.xlsx` - Sample balance sheet
- `cash_flow_sample.xlsx` - Sample cash flow statement

## Usage

These files can be uploaded through the VWB UI to test:
- Document parsing (Excel)
- Chart of accounts mapping
- Data normalization
- Validation rules
- Workbook generation
- Valuation calculations

## Note

The data in these files is synthetic/anonymized and does not represent any real company.
